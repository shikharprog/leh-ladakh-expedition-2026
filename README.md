# 🏍 Leh Ladakh Motorcycle Expedition 2026

A detailed day-by-day itinerary for a 10-day motorcycle expedition through 
Leh Ladakh — covering some of the highest motorable roads on Earth.

## 👥 The Crew
5 riders · 5 machines · one legendary road

## 🗺️ Route
Chandigarh → Srinagar → Leh → Nubra Valley → Pangong → 
Hanle → Umling La → Tso Moriri → Jispa → Manali → Dehradun
